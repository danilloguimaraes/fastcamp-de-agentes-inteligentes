# Descrição
Pydantic possui valor em serialização e validação. Se os requisitos não forem sobre isso, é melhor utilizar uma class. 
BaseModel é classe básica do Pydantic que identifica as regras de validação de dados de uma instância.
Evolução do uso básico dos tipos built-in do pydantic é a utilização de Field_Validator. 
Mode: define quando o valor será aplicado. Before or After.
